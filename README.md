# Phylosaur

A daily phylogenetic classification challenge focused on non-avian dinosaurs.
Inspired by [Metazooa](https://metazooa.com/).

## How to Play

Each day a mystery dinosaur is selected. Submit genus names and the 
phylogenetic tree progressively reveals shared ancestry between your 
guesses and the target. The closer your guess on the tree, the more 
of the lineage is revealed.

Five difficulty levels based on Wikipedia pageview popularity, from 
well-known genera to obscure taxa.

## Data Sources

- Genus list: [Paleobiology Database](https://paleobiodb.org)
- Taxonomic lineages: [Wikispecies](https://species.wikimedia.org)
- Difficulty ratings: Wikipedia Pageviews API
